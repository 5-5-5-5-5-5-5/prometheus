# Documentação

Este diretório agrega guias operacionais, arquitetura e recursos para contribuir e operar o Prometheus.

## Índice rápido

- **Guias**: [docs/guias](docs/guias) — Início rápido, comandos e configuração.
- **Arquitetura**: [docs/arquitetura](docs/arquitetura) — segurança, type-safety, mensagens e erros.
- **Checklist de Issues**: [docs/ISSUES-CHECKLIST.md](docs/ISSUES-CHECKLIST.md) — como abrir issues úteis e templates.

## Como usar

- Leia o [Guia de Início Rápido](docs/guias/GUIA-INICIO-RAPIDO.md) para executar o Prometheus localmente.
- Consulte [GUIA-COMANDOS.md](docs/guias/GUIA-COMANDOS.md) para descobrir subcomandos e flags.
- Consulte [docs/arquitetura](docs/arquitetura) para entender decisões de design.

## Manutenção da documentação

Se você atualizar guias, por favor faça um PR que inclua: resumo das mudanças, arquivos alterados e um teste rápido (comandos usados).
